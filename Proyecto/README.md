-Nombre del proyecto: Codificación de Huffman.

-Objetivo: Implementar el algoritmo de Huffman en Haskell para comprimir y descomprimir cadenas de texto así como implementar las listas y árboles dentro del proyecto. 

-Ejecución: Para poder ejecutar el proyecto es necesario tener instalado GHC. 
Para ejecutar se necesita abrir una terminal y ubicarse en la carpeta donde están los archivos Aux.hs y Proyecto.hs, depués vamos a iniciar el intérprete de Haskell con la siguiente instrucción: ghci Proyecto.hs
y ahora vamos a ejecutar el ejemplo con el siguiente comando: ejemploAlgorit
Para salir del intérprete usamos el siguiente comando: :quit
